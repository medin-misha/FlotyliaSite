# FlotyliaSite

Сайт и CRM система для Пражской Флотилии доставки.

## Бэкенд

Бэкенд находиться в папке `backend/` представляет собой FastAPI приложение. Основной стек:

- FastAPI
- PostgreSQL
- Асинхронная SQLAlchemy

Модели базы данных описанны в файле Architecture.canvas (открывать obsidian-ом)
Вся информация по бэкенду в папке backend/README.md.
